Welcome to GoalRadar

GoalRadar is a mini Dream11-style football web application where you can explore players, analyze their performance, and build your own dream team.

Features

  • Fetch player data using API
  • Radar chart with 6 performance factors
  • Create your own Dream11 team
  • Responsive and interactive UI

About This Project

This project is built using JavaScript and API integration using fetch(). It demonstrates dynamic data handling, visualization, and user interaction.
