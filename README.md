# ⚽ Pro Football 11 - Team Builder

## 📌 Project Overview
Pro Football 11 is a web application that allows users to build their own football team by selecting players and placing them on a football pitch using different formations.

The project demonstrates the use of JavaScript, API integration, and interactive UI development.

---

## 🚀 Features

- 🎯 Multiple Formations (15+)
- ➕ Add players to pitch using "+" slots
- ❌ Remove players by clicking again
- 🔍 Search players by name
- 🎛️ Filter players by position (GK, DEF, MID, ATT)
- 📊 Sort players by stats:
  - Overall
  - Pace
  - Shooting
  - Passing
  - Dribbling
  - Defending
  - Physical
- 🚫 Prevent duplicate player selection
- 📱 Fully responsive design (mobile + desktop)
- ⚠️ Error handling (API failure, no data, etc.)
